# Izza Study
Este projeto tem como objetivo desenvolver um bot para o Discord capaz de ensinar programação desde o básico, bem como gerenciamento de servidor no geral. O bot é projetado para ser uma ferramenta educacional e útil para os membros do servidor, oferecendo recursos e conhecimentos valiosos.

O bot incluirá recursos de ensino de programação em várias linguagens, bem como tutoriais e dicas sobre gerenciamento de servidor, como gerenciamento de canais, permissões, funções e outras funcionalidades do Discord.

Este projeto é de código aberto, portanto, sinta-se à vontade para contribuir com o desenvolvimento do bot, fornecer feedback, relatar problemas ou sugerir novos recursos. O objetivo final é criar um bot poderoso, educacional e útil para a comunidade Discord.
